# DISC Assessment Tool

A comprehensive DISC personality assessment tool that measures both **Natural** and **Adaptive** behavioral styles.

## Features

- **Dual Profile Measurement**: Captures both natural (comfort zone) and adaptive (under stress) behavioral styles
- **16 Research-Based Questions**: Each question measures MOST and LEAST like you
- **Visual Results**: Bar charts, radar charts, and shift analysis
- **Team Analytics Dashboard**: View team distribution, profile shifters, and department breakdowns
- **Export Functionality**: JSON export for integration with Notion or other tools

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the assessment.

### Production Build

```bash
npm run build
npm start
```

## Deployment

This project is optimized for deployment on Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

## DISC Profile Types

| Type | Name | Natural Style | Under Stress |
|------|------|---------------|--------------|
| D | Dominance | Direct, results-oriented, decisive | More forceful and demanding |
| I | Influence | Enthusiastic, optimistic, collaborative | More talkative and emotional |
| S | Steadiness | Patient, reliable, team-oriented | More passive and accommodating |
| C | Conscientiousness | Analytical, detail-oriented, systematic | More critical and withdrawn |
